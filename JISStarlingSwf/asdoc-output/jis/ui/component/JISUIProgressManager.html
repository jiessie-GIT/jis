Classpublic class JISUIProgressManager
InheritanceJISUIProgressManager JISUIManager starling.events.EventDispatcher

管理UI中的进度条组件，UI需要具备命名为_Center的Display，当设置进度的时候会改变_Center的width或者height属性，建议_Center为Scale9Image 可选组建_Text类型为TextField，存在该组件的话，会在设置进度的时候赋值“当前进度/最大进度”
